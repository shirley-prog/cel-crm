# Cél CRM
### a maneira mais fácil de fazer vendas

## Estrutura do projeto

```
cel-crm/
├── public/
│   └── index.html
├── src/
│   ├── index.js
│   └── App.jsx
├── package.json
├── .gitignore
└── README.md
```

## Como publicar no Vercel

1. Faça upload de todos esses arquivos no seu repositório GitHub
2. Acesse vercel.com e clique em "Add New Project"
3. Selecione o repositório `cel-crm`
4. Em **Framework Preset** selecione **Create React App**
5. Clique em **Deploy**
6. Aguarde ~2 minutos — o link estará pronto
